# JAQU-CAZ-Ruby-Common

A repository for housing shared/common components used by JAQU CAZ UIs.

## Installation

Add this line to your application's Gemfile:

```ruby
git 'git@github.com:/InformedSolutions/JAQU-CAZ-Ruby-Common' do
  gem 'custom_logger'
end
```

And then execute:

    $ bundle install
